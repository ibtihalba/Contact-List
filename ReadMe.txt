# Contact Management System

## Description

The Contact Management System is a simple C++ console-based application that allows users to manage their personal or professional contacts.
 Users can perform various operations such as adding, editing, deleting, searching, and listing contacts. 
 Additionally, contacts can be saved to and loaded from a file for persistent storage.


## Features

- **Add Contact**: Add a new contact by providing a name and a valid phone number.
- **Edit Contact**: Update the phone number of an existing contact.
- **Delete Contact**: Remove a contact by name.
- **Search Contact**: Search for a contact by name or phone number.
- **List All Contacts**: Display all contacts in the list.
- **Save and Load**: Save contacts to a file for future use or load them when the application starts.
